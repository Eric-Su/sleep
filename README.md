# sleep
sleep command line tool

Sleep v0.2
Usage: sleep [-options] n[d | h | m | s]
       default n milliseconds to pause
where options include:
       -h | --help     print this help message
       -v | --verbose  enable verbose output
Example:
       sleep 900
       sleep 1h-0.99h
       sleep 3s0.2m .1h
